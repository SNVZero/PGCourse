Пример PostgreSQL запроса

Следующий запрос демонстрирует использование оконных функций, CTE и агрегатных функций в PostgreSQL:

WITH sales_data AS ( SELECT region, product_category, sale_date, SUM(amount) AS daily_sales FROM sales WHERE sale_date BETWEEN '2023-01-01' AND '2023-12-31' GROUP BY region, product_category, sale_date ), ranked_sales AS ( SELECT region, product_category, sale_date, daily_sales, RANK() OVER ( PARTITION BY region, product_category ORDER BY daily_sales DESC ) AS sales_rank, AVG(daily_sales) OVER ( PARTITION BY region, product_category ) AS avg_category_sales FROM sales_data ) SELECT region, product_category, sale_date, daily_sales, avg_category_sales, ROUND((daily_sales / avg_category_sales - 1) * 100, 2) AS percent_above_avg FROM ranked_sales WHERE sales_rank = 1 ORDER BY region, percent_above_avg DESC;

Этот запрос анализирует продажи по регионам и категориям продуктов, определяя дни с максимальными продажами в каждой категории и вычисляя, насколько эти продажи превышают средние показатели.
