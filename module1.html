Модуль 1: Основы PostgreSQL и PostgreSQL

Введение в реляционные базы данных, установка PostgreSQL, базовые операции SELECT, INSERT, UPDATE, DELETE.

Введение в PostgreSQL

PostgreSQL — это мощная, открытая объектно-реляционная система управления базами данных (СУБД), известная своей надежностью, соответствием стандартам и богатым набором функций.

Установка и настройка PostgreSQL

PostgreSQL можно установить на различные операционные системы. Вот основные шаги:

# Для Ubuntu/Debian sudo apt update sudo apt install postgresql postgresql-contrib # Запуск службы PostgreSQL sudo systemctl start postgresql # Подключение к серверу sudo -u postgres psql

После установки создайте свою первую базу данных:

-- Создание базы данных CREATE DATABASE mydatabase; -- Подключение к базе данных \c mydatabase;

Основные PostgreSQL команды

PostgreSQL (Structured Query Language) — язык структурированных запросов для работы с базами данных.

Команда SELECT

Оператор SELECT — ключевой инструмент в PostgreSQL для извлечения данных из базы данных. Его задача — сформировать набор результатов (результат‑сет) — таблицу с выбранными данными, которая возвращается пользователю.

SELECT * FROM employees;-- Выбрать все столбцы из таблицы
SELECT first_name, last_name, salary FROM employees;-- Выбрать конкретные столбцы
SELECT DISTINCT department FROM employees; -- Выбрать уникальные значения

Изменение данных (INSERT, UPDATE, DELETE)

Оператор INSERT

Оператор INSERT в PostgreSQL предназначен для добавления новых записей (строк) в таблицу базы данных. Это одна из ключевых команд языка DML (Data Manipulation Language), позволяющая наполнять таблицы данными. Суть работы оператора заключается в том, что он формирует новую строку в указанной таблице, заполняя её заданными значениями в соответствии со структурой таблицы. При этом строго соблюдается соответствие между типами данных столбцов и передаваемыми значениями, а также учитываются ограничения (NOT NULL, уникальные ключи, внешние ключи и др.), определённые при создании таблицы.

INSERT INTO employees (first_name, last_name, department, salary) VALUES ('Иван', 'Петров', 'IT', 75000);-- Добавление одной записи
INSERT INTO employees (first_name, last_name, department, salary) VALUES ('Мария', 'Сидорова', 'HR', 65000), ('Алексей', 'Иванов', 'Sales', 70000); -- Добавление нескольких записей

UPDATE - обновление данных

Оператор UPDATE в PostgreSQL предназначен для изменения уже существующих данных в таблице базы данных. Это один из ключевых инструментов языка DML (Data Manipulation Language), позволяющий корректировать значения в отдельных столбцах или целых строках в соответствии с заданными условиями. Суть работы оператора заключается в том, что он находит строки, соответствующие определённому критерию, и заменяет в них значения указанных столбцов на новые. При этом система строго следит за соблюдением ограничений целостности данных: типов столбцов, правил NOT NULL, уникальных и внешних ключей. Если обновление нарушает какое‑либо из этих правил, операция прерывается с ошибкой.

UPDATE employees SET salary = salary * 1.1; -- Обновление всех записей, увеличить зарплату на 10%
UPDATE employees SET department = 'Engineering' WHERE department = 'IT'; -- Обновление с условием

DELETE - удаление данных

Оператор DELETE в PostgreSQL предназначен для удаления строк из таблицы базы данных. Это ключевая команда языка DML (Data Manipulation Language), позволяющая выборочно или полностью очищать таблицы от данных. Суть работы оператора заключается в том, что он идентифицирует строки, соответствующие заданному условию, и безвозвратно удаляет их из таблицы. При этом структура таблицы (её столбцы, ограничения, индексы) остаётся неизменной — изменяются только содержащиеся в ней данные.

DELETE FROM employees WHERE department = 'HR';-- Удаление с условием
DELETE FROM employees;-- Удаление всех записей (осторожно!)

  3. Измените должность одного сотрудника
